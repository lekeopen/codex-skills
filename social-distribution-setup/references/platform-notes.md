# Platform Notes

## Facebook Pages

Use the Make Facebook Pages module for page posts. A custom Facebook App is only needed if the user cares about the visible app/source branding.

## LinkedIn

Use a company page connection, not a personal profile, when publishing official brand updates.

## X

Prefer Buffer as the first implementation path. Direct X API access may require paid developer access and can change frequently.

## Email

Email is useful as an internal notification or archive. It should not be treated as proof that social posting succeeded.

## Out of Baseline Scope

WeChat Official Account, Xiaohongshu, Douyin, and similar platforms often need stricter account permissions, IP whitelists, content review, or manual publishing. Treat them as separate platform projects.
