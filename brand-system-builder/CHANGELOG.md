# Changelog

All notable changes to Brand System Builder are recorded in this file.

## [1.0.0] - 2026-07-19

### Added

- Initial production release of Brand System Builder.
- Approved brand lifecycle Scope.
- Approved `N`, `U`, `W`, and `U+W` workflow paths with Stages 00–09 and Approval Gates G0–G9.
- Cross-stage validation, system freeze, delivery freeze, governance, and change-control rules.
- Approved test scenarios for new brands, brand upgrades, website adoption, and combined upgrade-and-adoption work.
- Installation, usage, version, and release documentation.

### Release constraints

- No Scope changes.
- No Workflow changes.
- No Test Scenario changes.
- No `SKILL.md` changes.
- No new product functionality or redesign.
